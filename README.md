Equipment Checkout System (C#)

A console-based Equipment Checkout System built in C# that simulates a real-world lending workflow (e.g., university or IT department equipment management).

Designed using object-oriented principles, layered architecture, and interface-based abstractions to emphasize clean code and separation of concerns.
